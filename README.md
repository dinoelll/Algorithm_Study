## 스터디 규칙

#### .md(마크다운) 문서 작성 규칙
- 파일의 시작은 무조건 # 제목 입니다.
- 파일 안의 소제목은 ### 소제목 입니다.

#### Pull Request

- [Programmers] - [본인 이름] 
- [Algorithm] - [InsertionSort] insertion_sort_ma.md

#### Commit Message

#Docs: 문서를 추가, 수정 하는 경우 
- Docs : add week01 files
- Docs : add insertion_sort_ma.md
  
#Rename: 파일 혹은 폴더명을 수정하거나 옮기는 작업만인 경우

#Remove: 파일을 삭제하는 작업만 수행한 경우
